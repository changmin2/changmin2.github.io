# python 반복문

반복할 구문이 많을 때 사용
i는 시작 번호 10은 끝번호

1. for 문

```python

for i in range(10):
  print("Hello")
```

2. while 문

```python
i=0
while i<3:
  print(i)
  i=i+1
```
